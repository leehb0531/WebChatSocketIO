# WebChatSocketIO
simple practice with socket.io

1. Open "Command Prompt"
2. In the prompt, go to the directory where you downloaded  ./socketIO
3. type "node server"
4. Open Chrome
5. Type "localhost:3000" on your Chrome Address bar
